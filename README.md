# 🎓 Student Dropout Prediction System

## 📌 Project Overview

Student dropout is a major challenge faced by educational institutions worldwide. Early identification of at-risk students can help institutions provide timely interventions such as academic support, mentoring, and financial assistance.

This project develops an end-to-end Machine Learning system that predicts whether a student is at high risk of dropping out using Decision Tree Classification and provides predictions through an interactive Streamlit web application.

---

## 🎯 Problem Statement

Predict whether a student is likely to:

* Continue and Graduate
* Drop Out

The objective is to identify students who require early intervention and support.

## 📈 Model Performance

* Accuracy: 81.13%
* Precision (Dropout): 70%
* Recall (Dropout): 72%
* F1 Score (Dropout): 71%

### Confusion Matrix

* True Negatives: 514
* False Positives: 87
* False Negatives: 80
* True Positives: 204

The model successfully identifies approximately 72% of students who are at risk of dropping out.
